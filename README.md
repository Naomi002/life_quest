# Life Quest

**Life Quest** is a mobile RPG built with Flutter that turns real-life goals into a game. Complete daily quests to gain XP, level up your character, and defeat the "Procrastination Monster."

## Features
* **XP System:** Earn experience for fitness, social, and study goals.
* **Boss Battles:** Interactive combat mechanics for high-priority tasks.
* **Persistent Progress:** Stats are saved locally so your hero never loses progress.
* **Material 3 Design:** A modern, dark-themed UI built for high-end mobile devices.

## Tech Stack
* **Framework:** Flutter
* **Language:** Dart
* **Storage:** SharedPreferences
